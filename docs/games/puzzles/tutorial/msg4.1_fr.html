Les tirs se déroulent de la même façon que les contrôles, mais peuvent être effectués à distance. Voyons si vous êtes capable de contrôler l'astronef rouge...
Pour cela, il vous faudra :

  1. Détruire le premier char.
  2. Neutraliser le second char.
  3. Détruire la première tourelle de l'astronef rouge (avec votre Vedette et le gros tas). *
  4. Contrôler le second char.
  5. Détruire la dernière tourelle.
*: pour cette action, il est possible que vous ayez besoin de lire cette section des règles.
