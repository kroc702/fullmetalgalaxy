You can destroy opponent units in the same way as controls, but can be done at higher range Let see if you are able to control the red freighter...
To do so, you will need to:

  1. Destroy the first tank.
  2. Neutralize the second tank.
  3. Destroy the first freighter turret (with your entry and the big pile). *
  4. Take control of the second red tank.
  5. Destroy the last turret.
*: For this action, it is possible that you need to read This section of the rules.
